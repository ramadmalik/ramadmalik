I set a point on grass hoper and attached it to a 4 sided polygon.

I then used a Deconstruct BREP function then I used the Curve-Midpoint function and attached it onto the B-REP function, giviing me the midpoints on each side. Using the Length function, I attached it to the edges on the BREP function and a attached Divison Function. Then I connected the outcome of the Divison Function onto the Scale Function which is connected to the Curve-Midpoint function.

I attached an extrude function onto the scale function's geomtry, and then I connected a vector function to an extrude function; Extruding from the midpoint of the sides and the center point

Using a boolean toggle and attaching it to the vector function, I set it to True which set the lengths of the vectors to one.

Using the amplitude function, I used it to change the length of the vector I'm using for the slots. I set the value at around 0.9 inches.

Using the move function, I created another vector function and connected it via the motion function.

I then procceded to use the Trim with Regions functions to cut the slits out of the polygon.
